= Accesos múltiples en SnapDrive para UNIX
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive para UNIX admite accesos múltiples. SnapDrive para UNIX no admite el uso de alias para dispositivos, creados mediante el `mknod` y/o. `rendev` comandos.
