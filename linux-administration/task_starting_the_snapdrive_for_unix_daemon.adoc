= Iniciar el daemon SnapDrive para UNIX
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Debe iniciar y ejecutar el daemon SnapDrive para UNIX antes de poder utilizar cualquier comando de SnapDrive para UNIX.

Debe iniciar sesión como usuario raíz.

.Pasos
. Inicie el daemon:
+
`*snapdrived start*`
